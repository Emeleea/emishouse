About

Emilia House Harary (she/her) is a community worker who lives in Toronto, Ontario. This website is a passion project of hers' in hopes of pointing newly transitioning queer people in the right direction regarding resources.

Layout heavily inspired by the Melee Library website.

This is a simple website template published with Github Pages. Fork this repository to create your own site!
