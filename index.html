About

Emi's House is a public list aiming to point to all vital Queer/Trans resources, services, and organizations in the Golden Horseshoe Area (Southern Ontario). This project was created by Emilia House Harary (She/Her), who is a community worker at George Brown College. She specializes in Queer/Trans resource collection and supports. This is a simple website template published with Github Pages. Fork this repository to create your own site!
